# Простой голосовой чат

Голосовой чат для общения с друзьями через браузер. Работает на GitHub Pages с HTTPS.

## Как использовать
1. Откройте чат по ссылке
2. Введите имя и название комнаты
3. Разрешите доступ к микрофону
4. Отправьте ссылку другу
5. Говорите!

## Особенности
- Полностью бесплатный
- Автоматический HTTPS от GitHub
- Видно, кто онлайн
- Можно включать/выключать микрофон
